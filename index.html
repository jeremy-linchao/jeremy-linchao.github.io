git忽略已经被提交的文件

发表于 2017-04-21   |   分类于 git

现在项目的根目录放了 .gitignore 文件，并且git远程仓库的项目根目录已经有了 logs文件夹, 但是之后想忽略日志里的内容，光改写.gitignore并不够，正确做法是：

  1. git rm –cached logs/xx.log
  2. 然后更新 .gitignore 忽略掉目标文件
  3. 最后git commit -m “We really don’t want Git to track this anymore!”
