  13. react渲染多个元素的时候需要加一个唯一的key值

  14. react的setState是一个异步的函数，调用完setState不能确保state值马上就会被更新，如果需要依赖前一个值来更新，比较好的做法是传一个回调函数作为参数，这个回调函数是在setState调用完成后执行的

mysql的中文处理

发表于 2017-02-08   |   分类于 mysql

mysql默认的字符编码可以通过show variables like ‘character%’ 来查看，

client 为客户端使用的字符集。
connection 为连接数据库的字符集设置类型，如果程序没有指明连接数据库使用的字符集类型则按照服务器端默认的字符集设置。
database 为数据库服务器中某个库使用的字符集设定，如果建库时没有指明，将使用服务器安装时指定的字符集设置。
results 为数据库给客户端返回时使用的字符集设定，如果没有指明，使用服务器默认的字符集。
server 为服务器安装时指定的默认字符集设定。
system 为数据库系统使用的字符集设定。

  1. sudo cp /usr/local/mysql/support-files/my-defalut.cnf /etc/my.cnf

  2. 停掉mysql ， sudo vim /etc/my.cnf ，加入

    [mysqld]

    character-set-server=utf8
    collation-server=utf8_general_ci

    [client]

    default-character-set=utf8

  3. ALTER DATABASE database_name CHARACTER SET utf8 COLLATE utf8_general_ci

（或通过sequel在database中）

  1. ALTER TABLE table_name CONVERT TO CHARACTER SET utf8 COLLATE utf8_general_ci(或通过sequel在table info中)

css

发表于 2017-02-05   |   分类于 html/css

  1. css也可以注释，使用/ /

  2. color代表前景色，background-color代表背景色

  3. css中对同一个选择器的多个声明，最后一个生效，这个特点可以用来作浏览器兼容，比如在color上，第一行写一个所有浏览器都兼容的color表示法，下面一行写一个较新的表示法

  4. 一个html元素可以加上多个css类，但最好限制在4个最多

  5. 盒模型的width和height

    1) 用像素表示： 表示盒子的精确大小

    2）ems: 相对于盒子里面文字的倍数

    3）百分数：相对于父盒子的比例

  6. 当margin:auto被用来做居中时，必须同时加上width这个属性用来标明该元素的宽度，否则这个div的宽度就被设置成它父元素的宽度了，也就没有居中的说法了

  7. display属性有如下几个： inline,block,incline-block,none

  8. 默认盒模型有一个缺点，就是padding和border会增加width，因此无法计算每个元素的实际大小

  9. 盒模型使用box-sizing这个属性指定的，默认值为content-box，这个模型的width并不是实际渲染的大小，实际元素为加上boder的宽度以及pading，为了解决这个问题可以把box-sizing改为border-box，在这个模型下，width就是实际渲染出来的大小，padding和border会被相应地减掉

  10. position用来指定元素的位置，有如下四个取值

    1）static： 默认值

    2）relative：这个将元素设定为相对，因此还需要指定top,left,bottom和right属性，表示相对于四周的像素值

    3）absolute: 这个元素表示该元素就被固定在那边，其他元素会将他视而不见

    4) fixed: 会固定住这个元素，即使在有滚动条的情况下

  11. 当一个元素是float时，在同一个父元素下的其他元素会靠着它流动起来，同时float的元素必须指定宽度
