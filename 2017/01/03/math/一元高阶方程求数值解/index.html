一元高阶方程求数值解

发表于 2017-01-03   |   分类于 math

今天同事给我一个求增长率的问题，比如年底时候月消费额是a，老板定了个指标一季度的消费额要到b，求每月的增长率是多少，抽象成数学问题就是

搜了很多，解决方案是通过python的sympy库

from sympy.solvers import solve

from sympy import Symbol

x = Symbol(‘x’)

solve(x 3+x 2+ x -4.054, x)
