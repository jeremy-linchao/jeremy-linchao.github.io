前端发ajax请求的方法

发表于 2017-02-10   |   分类于 ajax

因为react本身只是个ui组件库，并不包含ajax请求的方式，所以有几个方法可以选择

  1. jquery： jquery因为大家都比较熟所以使用简单，但由于jquery比较大，如果单纯只用它这个功能不太合适
  2. superagent： 使用过几次，支持promise，可以工作在浏览器和node
  3. axio：没有使用过，promise_based，可以工作在浏览器和node
  4. request： 使用的最多的，但本身不支持promise
  5. fetch api: 这是html5的api，不是ecma的，node有node-fetch版本
