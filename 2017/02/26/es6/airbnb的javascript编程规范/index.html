airbnb的javascript编程规范

发表于 2017-02-26   |   分类于 javascript

  1. 对于所有引用类型使用const，这样可以避免对类型变量重新赋值，注意const类型也是块级作用域的

  2. 创建对象尽量使用字面值常量语法

  3. 对象的属性和方法使用简写方法

  4. 定义对象属性值尽量不加引号，除非有-这种字符

  5. 使用扩展运算符来copy对象，如果忽略对象时使用剩余运算符

  6. 使用push方法来向数组添加元素，使用扩展运算符来copy数组

  7. 使用对象解构来访问对象的多个属性

  8. 常量字符串使用单引号而不是双引号

  9. 如果要通过编程来构建字符串，使用模板字符串

  10. 如果需要判断函数参数
