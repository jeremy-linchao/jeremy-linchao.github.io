发表于 2017-04-12   |   分类于 database

yum 安装

  1. yum install -y mysql-server

  2. chkconfig mysqld on

  3. vim /etc/my.cn, 加入utf8字符集并修改下datadir，其他高级配置慢慢研究

  4. service mysqld start 启动mysql

  5. /usr/bin/mysql_secure_installation 设置root密码等安全特性
