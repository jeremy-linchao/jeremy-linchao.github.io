go学习

发表于 2017-09-19   |   分类于 go

  1. go的代码组织： 所有代码都在一个workspace下面，一个workspace下面有很多个仓库，代表不同项目，不同项目下有自己的package，package是源代码

  2. go的workspace下有三个跟目录， bin放编译好的二进制，pkg放依赖的包，src放不同项目的源文件

  3. GOPATH是用来设置go的workspace的目录，这个不要跟go的安装目录混淆，并且这两个也不能在一个目录下，go的安装目录默认在/usr/local, 因此在~/.zshrc下面要设置下GOPATH

    另外正如前面所说，所有编译好的目录都在go的workspace下，因此也可以把GOPATH下的bin加到PATH环境变量下

  4. Go的package都是静态链接的，因此运行时是不需要库存在的

  5. go的第一行必须是package name，可执行的命令必须用package main

  6. 用go get获取远程仓库的package

语法

  1. 大写字母开头的成员才会被导出

  2. 类型是写在变量后面，函数可以返回任意数量的结果

  3. 返回值也能被命名，尽量不要用naked return（返回值的名字是写在函数定义行里）

  4. 在函数内部可以用 := 来声明变量，这里可以省去var关键字以及类型，当然这个只适用于函数内部

  5. 变量在定义时如果没有初始化则会给一个0值， 具体是值

    如果是数值类型，默认是0， 如果是bool类型，默认是false，如果是字符串类型，默认为”“

  6. 强制转化的语法是T(v) ,

    go里没有隐式转换

  7. 常量用const关键字，并且常量名首字母大写

  8. 只有for循环，for虚幻的初始化，判断条件不需要括号，执行体的花括号还是需要的

  9. if也不需要括号

  10. go的switch只要命中case就中止，不会继续往下走，因此就不需要break了

  11. 没有条件的switch可以用来代替长if else

  12. defer语句：A defer statement defers the execution of a function until the surrounding function returns.
