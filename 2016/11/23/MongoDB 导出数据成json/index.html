MongoDB 导出数据成json

1. mongoexport -d ticketFinder -c airports –fields airportName,airportCode –jsonArray –pretty –out ~/GitHub/ticketFinder/frontend/airportscode.json

这里 -d 指的是数据库名， -c指的是collection，- -filed用来指定字段， - -jsonArray可以把输出变成一个json数组，而不是那种一个对象一行的形式，这里要注意的是如果输出是json格式，id这个字段也会被保留输出
