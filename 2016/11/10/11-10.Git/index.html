Git

Git 使用

  1. 首先git clone 主分支
  2. 下载到本地之后checkout 一个新的分支 git checkout -b from_linchao
  3. 更新代码后git push

Git 中的命令理解

  1. git remote add 将GitHub上创建的仓库设为本地仓库的远程仓库
  2. git diff 该命令是用来比较工作目录和staging are的区别，结果是说你改了什么东西，但你没有git add加进到staging区域，因此如果你所有修改都git add了，git diff是没有输出的
  3. git diff - - staged 比较你staged的东西和你上一次提交的区别
