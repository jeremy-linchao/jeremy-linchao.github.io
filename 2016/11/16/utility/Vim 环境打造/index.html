Vim 环境打造

发表于 2016-11-16   |   分类于 utility

1. 思路是使用spf13来配置vim，https://github.com/spf13/spf13-vim，可以节省比较多的时间

2. 几个常用插件使用方式

  • 注释和取消注释： ，+ c + space

  • 查找文件： ctrl + p ，然后使用 ctrl + j,k 来上下移动

  • 打开工程目录： ctrl + e, 然后使用t来打开文件，在左边和右边窗口移动使用ctrl + H，L来左右移动鼠标，在多个tab之间使用gt来切换

  • 颜色配置：在.vimrc那个文件找到solarized那一段，let g:solarized _termcolors = 16 这样的话文本不会太暗

  • 字体配置： 首先把let g:airline_powerline_fonts=1这段写到.vimrc.before.local，同时在终端的perference上配置下字体为SourceCodePro

3.暂时不用的插件先都关闭掉

  • echo UnBundle \’AutoClose\’ >> ~/.vimrc.bundles.local 把不需要的插件写到.vimrc.local，然后:BundleClean! 移除插件目录
