函数式编程

发表于 2016-12-21   |   分类于 coding

有些事物在你得到之前是无足轻重的，得到之后就不可或缺了

  1. 纯函数：相同的输入，永远得到相同的输出，而且没有任何可观察的副作用
  2. 副作用：在计算结果的过程中，系统状态的一种变化，或者与外部世界进行可观察的交互
  3. 函数的柯里化：只传递给函数一部分参数来调用它，让它返回一个函数去处理剩下的参数
  4. currying : the technique of translating the evaluation of a function that takes multiple arguments into evaluating a sequence of functions, each with a single argument.
  5. partial application : the process of fixing a number of arguments to a function, producing another function of smaller arity
  6. 柯里化和偏函数的区别：区别在于固定的参数不同，柯里化每次固定一个参数，返回一系列一元函数的调用
