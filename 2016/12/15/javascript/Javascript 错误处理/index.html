Javascript 错误处理

1.在if语句里很容易发生类型转换错误，比如

这个函数的意思是对字符串做连接，如果第三个字符串不为空，就加上，如果为空就不加，这个本意是好的，但是javascript因为不是强类型的，在函数的参数里是不检测参数类型，所以你不知道会传进来个什么参数，所以如果传个数值1进来，就会被转化成true，进而加上，所以这里最好的做法是检测str3的类型是不是字符串 typeof str3 === “string”

对于基本类型应该使用typeof来检测，对象类型使用instanceof，所有面向公众的API应该无条件执行类型检查

  1. 在发ajax请求时对于url传的查询参数一定要进行编码，使用encodeURIComponent

  2. 使用try catch

    模块初始化使用循环，如果一个模块出问题就会阻断程序运行，使用try-catch来解决这个问题
